# 🛍️ SQL-Based Online Hijab Store Database System

This project focuses on designing and implementing a relational SQL database system for an Online Hijab Store. The system was developed to manage key business operations such as product management, inventory control, customer records, order processing, payments, shipping, suppliers, materials, returns, and product reviews.

The project involved creating a complete database design starting from the flowchart, Entity Relationship Diagram (ERD), and Extended Entity Relationship Diagram (EERD) to clearly define the system structure, business rules, entities, attributes, and relationships. The database was then implemented using SQL by creating tables, defining primary keys and foreign keys, inserting sample records, and performing Data Manipulation Language (DML) operations.

The system also applied normalization up to 3NF to reduce redundancy and improve data consistency. A data dictionary was prepared to document all entities and attributes used in the database.

🛠️ Tools Used: SQL, MySQL, draw.io

📌 Key Skills: Database design, ERD, EERD, normalization, primary key, foreign key, DML operations, data dictionary

🎯 Goal: To build a structured and efficient database system that supports smooth online store operations and improves data management for an e-commerce business.
